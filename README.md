# 🔳 Python QR Code Generator

A simple and practical QR Code Generator built with Python.

This project allows the user to enter a URL or text and generate a QR code image saved as a `.png` file. It was created as part of my Python learning process to practice external libraries, file handling, user input validation, and clean project structure.

---

# 📌 Project Overview

The Python QR Code Generator is a beginner-friendly terminal application that converts user input into a QR code image.

The user can enter any URL or text, choose a custom output file name, and the program automatically saves the generated QR code inside an `output` folder.

This project is useful for learning how Python can be used to create real files and work with third-party packages.

---

# ✨ Features

* Generate QR codes from URLs or text
* Save QR codes as `.png` images
* Custom output file name
* Automatic `output` folder creation
* Input validation
* Error handling
* Clean function-based structure
* Beginner-friendly terminal interface

---

# 🛠 Technologies Used

* Python 3
* `qrcode` library
* `Pillow` image support through `qrcode[pil]`
* Built-in `os` module

---

# 🚀 How to Run the Project

## Step 1 — Clone the Repository

```bash
git clone https://github.com/DonUserOn/python-qr-code-generator.git
cd python-qr-code-generator
```

## Step 2 — Install Requirements

```bash
pip install -r requirements.txt
```

## Step 3 — Run the Program

```bash
python main.py
```
